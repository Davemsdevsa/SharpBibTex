    • Sharpbibtex

    BibTex is a great way of handling references for academic work but the support in word is lacking. This project aims to resolve that problem by creating a BibTexParser and an Office Addin for Word that Processes BibTex references off of Academic Sites

    Project maintained by Davemsdevsa Hosted on GitHub Pages — Theme by mattgraham

    Welcome to SharpBibTex

    This project is aimed at helping developers integrate BibTex into their projects as well as project a free open source solution for academics to capture Bibtex references directly into MS Word 2016 and Higher.

    Check out the Wiki on GitHub for more details..
